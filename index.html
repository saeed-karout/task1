there are a few different background properties

this property is the location of the image itself,for example:

or

background-color

this allows us to add color to a background .An example is css would like like this.

background-repeat

this is how the image will repeat .the value can be any of the following

backgrpund-repeat :repeat;

this repeat on both the X and Y axis, and will assintially taile the image across entire
background

backgrpund-repeat :repeat-x;

this repeat backround on the X axis, or horaizonality

backgrpund-repeat :repeat-y;

this repeat backround on the Y axis, or vertically

backgrpund-repeat :no-repeat;

this show the image onle once, without repeting

background-position

this is the actual position of the background image within it's container.The values can be:

backgrpund-position :top;
backgrpund-position :right;
backgrpund-position :bottom;
backgrpund-position :left ;
backgrpund-position :center;

it can also be a percentag, length, or combinition, like so:

backgrpund-position : 50px 20%;
